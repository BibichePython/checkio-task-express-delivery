On the island our Robots have found the mysterious Boxes. After some examination Nicola have discover the interesting feature of these Boxes. If you place something in any of the Boxes, then you can take it from any Box. Stephan wants to use this property for delivery of cargo across of the island. He moves without loads twice faster than without it. So Stephan can place the cargo while going to delivery point and take it later in other box.

The map for delivery is presented as an array of strings, where:

  • "W" is a wall
  • "B" is a box
  • "E" is an goal point.
  • "S" is a start point.
  • "." is an empty cell.

Stephan move between neighbours cell in two minutes if he is loaded. Without it -- in one minute. Loading and unloading of cargo in (out) the box takes one minute. You should find the fastest way for cargo delivery (minimum time).

The route is a string, where each letter is an action.

  • "U" -- Up (north)
  • "D" -- Down (south)
  • "L" -- Left (west)
  • "R" -- Right (east)
  • "B" -- Load or unload in (out) a box.

Input: A map for delivery, a list of strings.

Output: The fastest route, a string.

Example: 

checkio(["S...","....","D.WD","..WE"]) #LLLDDD
checkio(["S...","....","D..D","..WE"]) #DDBLLLBD
